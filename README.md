## ✨ Key Features
- **🚀 Cinematic Animations:** "Split & Merge" scroll reveals across all sections using custom spring physics.
- **🌌 Advanced 3D Interactions:** Dynamic 3D tilt and parallax hover effects on project cards and skill tags.
- **🔮 Premium Aesthetic:** Deep dark theme featuring sleek Glassmorphism UI, floating pill navigation, and subtle radial glowing gradient accents.
- **📱 Mobile-First Design:** 100% fully responsive layout ensuring a perfect viewing experience on any device.
- **🧩 Modular Architecture:** Clean React component structure spanning Hero, About, Skills, Featured Projects (including a *Bank Transaction System* & *Social Feed App*), and Contact sections.

## 🛠️ Tech Stack
- **Frontend Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (Utility-first styling, no custom CSS)
- **Animations:** Framer Motion (Scroll reveal, 3D tilt, spring physics)

## 🚀 Quick Start / Installation

To run this portfolio locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in your browser:**
   Open `http://localhost:5173` to see the site running locally.
